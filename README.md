# cwl-pfb-unwrapper
Developed by Jared Rozowsky and improved by Cera Fisher 

### Contents of this repo
1. `manifest.py` is the Python script that creates the DRS manifest from the input directory of `tsv` files.
2. `pfb-unwrapper-hs.cwl` is the json export of the CWL app in the BDC project.
